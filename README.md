# Разработка интерактивной веб-платформы для игры в гибридные и модифицированные версии классических настольных игр

## Проектирование, программная реализация и развертывание адаптивной веб-платформы «Шахматы и Шашки Гибрид», обеспечивающей одновременное функционирование шахматных и шашечных механик на одной доске, с различными модификациями и настройками.

## Технологии
- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript (ES6)
